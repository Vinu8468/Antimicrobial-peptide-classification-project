# Antimicrobial Peptide Classification using Machine Learning

## Overview

This project explores the use of machine learning to classify **antimicrobial peptides (AMPs)** based on sequence-derived features.

Antimicrobial peptides are short amino acid sequences that play a key role in innate immunity by disrupting microbial membranes or interfering with intracellular processes.

The goal of this project is to:

* Understand the biological characteristics of AMPs
* Convert peptide sequences into numerical features
* Train machine learning models to distinguish AMPs from non-AMPs
* Evaluate model performance using standard metrics

---

## Objectives

* Explore AMP sequence data
* Perform feature extraction from protein sequences
* Build and compare classification models
* Interpret results in a biological context
